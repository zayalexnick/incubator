# Тестовое задание для GoodLine Инкубатор

Задание выполнено Зайка Александром

## Запуск

### webpack-dev-server
```
yarn start:dev
```
или
```
npm run start:dev
```

### Сервер
```
yarn server
```
или
```
npm run server
```

## Сборка
```
yarn build:prod
```
или
```
npm run build:prod
```

После сборки проекта создается директория /dist
Сервер при запросе на / откроет клиентскую часть. Все запросы отправляются на /api
